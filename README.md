# 记忆游戏

## 介绍

这是 Udacity 前端纳米学位进阶课程中的项目一：记忆游戏。

## 游戏规则

- 玩家通过点击来翻转卡片
  - 若两张翻转的卡片图案相同则保持打开状态
  - 否则重新盖住
- 玩家需要记忆卡片图形，直至所有卡片打开，方能赢得游戏胜利
- 玩家在赢得游戏后，会根据过程中点击次数来进行等级评价
